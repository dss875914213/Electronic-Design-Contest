# How to
1. 注册[github](https://github.com/)帐号，新建一个仓库X；
2. 本地计算机安装[git](https://git-scm.com/)，在本地项目根目录初始化仓库，连接github上的仓库X；
3. 将本地仓库推送至github端；
4. 克隆`刘老师`的仓库到本地，修改readme文件(加入仓库X的链接)，向刘老师提交修改申请(如果是建设者不需要fork操作)。
