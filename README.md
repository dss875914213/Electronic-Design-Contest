# 201-203项目团队的工作方式
1. 注册[GitHub](https://github.com/)帐号，并新建一个仓库（下文称其为仓库X）；
2. 本地计算机安装[git](https://git-scm.com/downloads)软件，通过命令行在本地的项目根目录(例如stm32项目根目录)初始化仓库，并连接远程仓库X；
3. 将本地仓库推送至github端；
4. 克隆`刘老师`的仓库到本地，修改readme文件(加入仓库X的链接)，向刘老师提交修改申请(如果是建设者不需要可以直接修改)。
