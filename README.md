# How to
1. 注册github帐号，新建一个仓库X

3. 将本地仓库推送至github端
4. 克隆`刘老师`的仓库，修改readme文件，加入到自己仓库的链接，向刘老师提交修改申请(如果是建设者不需要fork操作)
